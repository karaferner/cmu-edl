# Overview of cmu-edl

The cmu-edl package is a framework for processing, analyzing, and plotting electrochemical testing data that is standard in the CMU-EDL group.

## Structure
Classes:
* Experiment
* Polarization
* EIS (Electrochemical Impedance Spectroscopy)
* Cyclic Voltammetry
* Durability

* IO/util
* Cross analysis
* Plotting
